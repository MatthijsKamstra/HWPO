Strength

Dumbbell Bench

3 Sets:
:35 Max Rep Dumbbell Bench // RPE 7-8
Rest 1:30 Between Sets.

NOTES:
- If you did this last week, use the same weight.
- Perform these with a @ 2020 tempo (i.e. :02 down, no pause at the bottom, :02 up, no pause at the top). Don't game the :35, perform controlled reps for the entire working time.
- Use a load that feels like an RPE 7-8.

OPTION A.

Strict Pull-Up

4 Sets of Max Reps (Cap @ 12 Reps)
Rest 2:00 Between Sets.

NOTES:
- We are adding one set this week. If you did this last week, aim to complete more total reps than last week.
- This option is for those that either have strict pull-ups but are not ready to load them, or people who do not have strict pull-ups and will need to use a band.
- Use a band if you will get less than five reps on any given set.
- Leave one rep in the tank on each set.
- If you are using bands, use as few bands as possible, and aim for at least 8 reps.
